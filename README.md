# tp-2023-1c-EAB-MODEL

Librerias para la entrega:
 - cspec:
```
git clone https://github.com/mumuki/cspec.git
cd cspec
make
sudo make install
```
 - commons:
```
git clone https://github.com/sisoputnfrba/so-commons-library
cd so-commons-library
make debug
make install
```
